# Password-Generator
# 🔐 Neon Aura Password Generator

Welcome to the **Neon Aura Password Generator** — a modern, visually stunning, and highly interactive web application designed to generate secure and customizable passwords with an immersive user experience.

This project combines **strong JavaScript logic**, **clean UI/UX design**, and **dynamic animated backgrounds** to create a tool that is both functional and visually impressive.

---

## 🌟 Project Overview

The Neon Aura Password Generator allows users to create **secure, random passwords** based on their preferences such as length and character types. It enhances the traditional password generator concept by integrating:

* Real-time visual feedback
* Animated UI interactions
* Aesthetic background effects
* User-friendly controls

This makes it not just a utility tool, but also a **portfolio-level frontend project**.

---

## ✨ Key Features

### 🔐 Password Generation

* Generate strong and random passwords instantly
* Adjustable password length (8–32 characters)
* Option to include:

  * Uppercase letters (A–Z)
  * Lowercase letters (a–z)
  * Numbers (0–9)
  * Symbols (!@#$%^&*)

---

### 🧠 Smart Security Logic

* Ensures at least one character from each selected category
* Uses randomized selection for better security
* Shuffles characters to avoid predictable patterns

---

### 📊 Strength Indicator

* Real-time password strength analysis
* Visual progress bar with color transitions
* Strength levels:

  * Weak
  * Fair
  * Good
  * Strong
  * Very Strong

---

### 📋 Copy & History

* One-click copy to clipboard
* Displays recently generated passwords
* Keeps last 5 passwords for quick access

---

### 🌗 Dark / Light Mode

* Toggle between dark and light themes
* Smooth transition effects
* User preference saved using localStorage

---

### 🌌 Animated Background (WOW Factor)

* Custom neon particle animation using HTML5 Canvas
* Smooth motion and glowing effects
* Creates a futuristic, immersive feel
* Optimized for performance across devices

---

## 🎨 UI/UX Highlights

* Glassmorphism design (blur + transparency)
* Clean and centered layout
* Smooth animations and transitions
* Mobile-friendly responsive design
* Minimal yet modern color palette
